# Visual QA Checklist

- No raw markdown visible.
- No visible `**bold**`.
- No visible unprocessed `:::check`, `:::task`, `:::quiz`, `:::checklist`.
- No raw `.md` answer link on the user-facing lesson page.
- Quiz controls work with keyboard and mouse.
- Feedback appears after checking the quiz.
- Checklist items are clickable.
- Answer reveal works inline.
- Mobile layout remains readable.
- Headings hierarchy is sensible.
- Spacing is readable and not cramped.
- Back link is visible.
- Auth redirect behavior matches the route contract.
