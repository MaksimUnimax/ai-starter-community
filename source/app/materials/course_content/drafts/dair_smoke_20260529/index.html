Как разрабатывать с помощью ChatGPT и Codex

Курс состоит из 9 уроков. Он остаётся тестовой версией, но уже показывает полный учебный маршрут и тему deploy key внутри урока про Git.

Документы работают как память проекта.

4

Сначала восстанавливаем контекст, потом продолжаем работу.

5

Как обновляется документация во время работы

Документы обновляются после важных решений и результатов.

6

Зачем нужны ТЗ и roadmap

ТЗ говорит, что строим. Roadmap говорит, в каком порядке.

7

Git простыми словами

Как дать Codex право отправлять проект в GitHub: deploy key, публичный ключ, приватный ключ, Allow write access.

8

Как идёт один безопасный шаг разработки

User -> GPT -> Codex -> report -> GPT -> visual check -> docs update.

9

Что значит отчёт Codex

GPT объясняет отчёт, а пользователю не нужен глубокий технический разбор.
